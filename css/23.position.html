Position의 이해

기본 상태는 static 이며, 배치 시 작성 순서대로 배치된다.
한 번 정해진 위치는 절대로 변경할 수 없다.
만약 position을 relative로 바꾸면 배치는 그대로지만 이제부터 위치조정이 가능하다.
top, left, right, bottom과 z-index 속성으로 제어가 가능
position이 absolute라면 좌표를 이용하여 위치를 지정할 수 있다.
기존에 존재하는 화면 위를 덮으며 배치된다(overlay라 부름)
z-index 속성을 통해 같은 absolute끼리의 우선순위를 정할 수 있다
top, left, right, bottom 속성으로 위치 설정이 가능
position이 fixed라면 뷰포트(Viewport)를 기준으로 위치를 설정할 수 있다
따라서 문서가 이동해도 fixed로 설정된 위치는 변하지 않는다
흔히 말하는 사용자를 따라오는 요소들을 구현할 수 있다
